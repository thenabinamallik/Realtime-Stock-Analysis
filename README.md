# AI Stock Trading Assistant

## Requirements
Python 3.10

## Setup

Clone repo

git clone https://github.com/Gopinath-08/stock-ai-system.git
cd stock-ai-system

Create environment

py -3.10 -m venv venv

Install dependencies

venv\Scripts\pip install -r requirements.txt

Run the app

venv\Scripts\python -m streamlit run app.py
