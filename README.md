# AI Stock Trading Assistant

## Requirements
Python 3.10

## Setup

Clone repo
```
https://github.com/thenabinamallik/Realtime-Stock-Analysis.git
cd Realtime-Stock-Analysis
```
Create environment
```
py -3.10 -m venv venv
```
Install dependencies
```
venv\Scripts\pip install -r requirements.txt
```
Run the app
```
venv\Scripts\python -m streamlit run app.py
```
